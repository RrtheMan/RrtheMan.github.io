The Passo

Introduction

Passo with AI is a university project that I developed as part of an individual module. The goal was to recreate the classic board game Passo in an executable program that users can play on their own devices. What made this project unique was the integration of a machine learning component that allowed users to play against an AI opponent trained to learn and improve its gameplay.

The project combined both game development and AI training, giving me the opportunity to explore neural network models and apply them in a fun, interactive way. It reflects both my technical knowledge and my interest in artificial intelligence and software development.

The Program

The game was built using Python, with libraries such as PyTorch for building and training the AI, and NumPy for data handling. The user interface and game mechanics were also implemented in Python, resulting in a complete and user-friendly experience. Users can play the game against the AI, which uses a neural network to make decisions based on previous moves and patterns. The interface was designed to be clear and intuitive, making it easy for users to navigate through the game, even if they had never played Passo before.

The AI Component

A major feature of the project was developing an AI that could understand and play the game effectively. Using a neural network built with PyTorch, the AI was trained to recognize game strategies and make decisions accordingly. It provided a challenging opponent and demonstrated the practical application of machine learning in game development.
