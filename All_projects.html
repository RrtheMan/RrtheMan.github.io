FAB Website

This website was developed for a textile start-up business

It was designed to allow users to create accounts, place orders, and securely make payments. The goal was to provide a smooth and user-friendly experience for both customers and the business owners.

Technologies used include HTML, CSS, JavaScript, PHP, and MySQL. HTML and CSS were used for layout and styling, JavaScript added interactivity, PHP handled server-side logic and user authentication, and MySQL managed data. The site is also mobile-responsive and performance-optimized.

Passo

This program was created for a university project

This individual project focused on integrating the classic game Passo into an executable program that allows users to play the game on their own devices. The core objective was to design and implement an AI opponent using a neural network-based machine learning approach. The AI was trained to learn the rules and strategies of Passo, enabling it to play intelligently against human players.

The project was developed using Python, with libraries such as PyTorch and NumPy for building and training the neural network. The game logic and interface were also implemented in Python, resulting in a fully executable and user-friendly program.

Catan

This program was developed collaboratively with a team as part of a software development module, where we applied agile practices and version control tools like Git to manage our workflow.

This group project focused on integrating the classic board game Catan into an executable program that allows users to play the game on their own devices. The core objective was to design and implement the full game experience, including building a user-friendly interface (UI) and enabling gameplay against a basic AI opponent.

The project was developed using Python, with libraries such as Pygame to build the game's interface and manage user interactions. Both the game logic and UI were implemented in Python, resulting in a fully executable and interactive application.

Lovejoy’s Antique Evaluation Web Application

This project involved developing a secure web application for a local antique dealer as part of a coursework module. The goal was to deliver a minimum viable product (MVP) that allows customers to register, log in, and request antique evaluations—while prioritizing security due to the client’s competitive environment.

Secure user registration and login with a password policy and encrypted user data. Password management tools including strength feedback and recovery functionality. A “Request Evaluation” form for logged-in users to submit item details and contact preferences (phone/email). Photo upload capability to support image-based evaluations. An admin-only dashboard to view and manage submitted requests.
